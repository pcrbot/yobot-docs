# Windows 使用 cqhttp-mirai 部署

# 部署过程

本文为便携版部署方式

# 环境准备

# 安装依赖

安装 JRE (OpenJDK 11)

点击下载 (opens new window)，双击安装

# 部署 yobot

下载 yobot 便携版 (opens new window)

解压后双击“yobot.exe”启动服务

# 部署 mirai

先准备如下的目录结构

mirai\
├── libs\
├── config\
│   └── Console\
└── plugins\
    └── CQHTTPMirai\

下载 1.mirai-core (opens new window) 2.mirai-console (opens new window) 3.mirai-console-terminal (opens new window) 三件套并放在 .\libs 目录里

下载 CQHTTPMirai.jar (opens new window) 并放在 .\plugins 目录里

新建 Mirai 配置文件在 .\config\Console\AutoLogin.yml，修改配置文件如下（前面是QQ号，后面是密码）

plainPasswords:
  123456654321: example

新建 CQHTTPMirai 配置文件在 .\plugins\CQHTTPMirai\setting.yml，修改配置文件如下（注意修改 QQ 号）

# 要进行配置的QQ号 (Mirai支持多帐号登录, 故需要对每个帐号进行单独设置)
# 详细说明请参考 https://github.com/yyuueexxiinngg/cqhttp-mirai
# 如果要登录多个账号，或者一个账号连接多个后端，直接按照 yaml 格式增加字段即可
"1234567890":
  ws_reverse:
    - enable: true
      postMessageFormat: string
      reverseHost: 127.0.0.1
      reversePort: 9222
      reversePath: /ws/
      accessToken: null
      reconnectInterval: 3000

新建入口文件在 .\start-mirai.bat，内容如下

@echo off
title Mirai Console
java -cp "./libs/*" net.mamoe.mirai.console.terminal.MiraiConsoleTerminalLoader %*
pause

双击 start-mirai.bat 启动 mirai

部署完成
